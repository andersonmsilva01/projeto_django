# 🎓 Projeto Django — Sistema Escolar

Projeto desenvolvido como trabalho final do curso de Django.

## 📋 Sobre o Projeto

Sistema web para gerenciamento de alunos e cursos, desenvolvido com Django.

## 🚀 Funcionalidades

- ✅ Cadastro de alunos
- ✅ Listagem de alunos
- ✅ Formulário com validação
- ✅ Autenticação e login
- ✅ Testes automatizados

## 🛠️ Tecnologias

- Python 3.9
- Django 4.2
- SQLite
- HTML

## ▶️ Como executar
```bash
# Clone o repositório
git clone https://github.com/andersonmsilva01/projeto-django

# Entre na pasta
cd projeto_django/escola

# Crie o ambiente virtual
python3 -m venv venv
source venv/bin/activate

# Instale as dependências
pip install django

# Rode as migrations
python3 manage.py migrate

# Inicie o servidor
python3 manage.py runserver
```
